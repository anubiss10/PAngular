Bienvenido a la Aplicación de Gestión de Empleados

Esta aplicación te permite administrar la información de los empleados de tu empresa.

Características Principales:

  • Registro de empleados.
  • Eliminación de empleados.
  • Lista de empleados paginada.
  • Inicio de sesión con JWT.
  • Acceso restringido a ciertas funcionalidades con autenticación.

Instrucciones de Uso:

  1. Inicia sesión para acceder a todas las funciones.
  2. Utiliza la página de lista de empleados para ver y paginar la información de los empleados.
  3. Accede a la página de agregar empleado para registrar nuevos empleados.
  4. Utiliza la página de eliminar empleado para quitar empleados de la lista.
